# Lab 4.2: Optimization and Validation of Linear Regression

## Work Description
This laboratory work covers the development and statistical analysis of a **multivariate linear regression** model. The task involves predicting a target variable (car price) based on several input features (year of production, mileage, engine capacity).

The primary goals are to implement optimization algorithms (Analytical and Iterative) and perform a rigorous statistical validation of the model's reliability and significance.

## Mathematical Model
The model assumes a linear relationship between features $X$ and the target $y$:
$$\hat{y} = w_1 x_1 + w_2 x_2 + \dots + w_n x_n + b$$
The objective is to minimize the **Mean Squared Error (MSE)**:
$$MSE = \frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2$$

### Optimization Methods
1.  **Ordinary Least Squares (OLS):**
    The analytical solution found by setting the gradient of MSE to zero:
    $$\mathbf{w} = (X^T X)^{-1} X^T y$$
    This method is precise but computationally expensive for very large datasets due to matrix inversion.
2.  **Gradient Descent (GD):**
    An iterative algorithm that updates weights in the direction of the steepest descent:
    $$w_{new} = w_{old} - \eta \nabla MSE$$
    It is more scalable and forms the foundation for training neural networks.

## Statistical Validation
Beyond calculating errors, we must prove the model's validity:
*   **Coefficient of Determination ($R^2$):** Shows the proportion of variance explained by the model.
*   **F-test (Fisher Criterion):** Tests the overall significance of the model against a "null" model (where all parameters are zero).
*   **t-test (Student Criterion):** Evaluates the significance of each individual feature. In this lab, all features (year, mileage, engine) were found to be statistically significant.

## Implementation & Results
The notebook `4.2 Linear Regression Optimization.ipynb` uses the "Used Cars Catalog" dataset. Both OLS and GD converge to the same MSE value (~1.62e7). The visualization shows the predicted vs. actual prices with a 95% confidence interval, illustrating where the model is most accurate and where outliers exist.
