# Lab 4.6: Neural Network Model for Image Recognition (MNIST)

## Work Description
This laboratory work focuses on the implementation and training of an **artificial neural network (ANN)** from scratch using NumPy. The task is to build a classifier for the **MNIST dataset**, which consists of grayscale images of handwritten digits (0–9).

The objective is to understand the core components of deep learning: forward propagation, backpropagation, and gradient-based optimization.

## Mathematical Model
The network is a multi-layer perceptron (MLP) with the following structure:
*   **Input Layer:** 784 neurons (representing flattened $28 \times 28$ pixels).
*   **Hidden Layer:** 128 neurons with a **Sigmoid** activation function: $\sigma(z) = 1 / (1 + e^{-z})$.
*   **Output Layer:** 10 neurons with a **Softmax** activation function to output probabilities for each digit.

### Training Process
1.  **Forward Propagation:** Information flows from input to output layers to get predictions.
2.  **Loss Calculation:** Cross-Entropy loss is used to measure the difference between the predicted probabilities and the one-hot encoded ground truth.
3.  **Backpropagation:** Using the Chain Rule, gradients of the loss with respect to all weights and biases are calculated.
4.  **Weight Update:** Gradient Descent is applied to adjust the weights: $W = W - \eta \cdot \nabla W$.

## Implementation & Results
The notebook `2.7 mnist.ipynb` (used for Lab 4.6) demonstrates the network's training over 15 epochs.
*   **Accuracy:** The model achieves approximately **93.65% accuracy** on the test set.
*   **Visualization:** Training curves show a steady decrease in loss and increase in accuracy. Sample predictions display original handwritten digits with labels highlighting correct (green) and incorrect (red) results.
